# Magnit DP Manual v22 — GitHub Pages ready

Версия адаптирована для публикации через GitHub Pages.

## Публикация

1. Загрузите **содержимое этой папки** в корень репозитория.
2. В GitHub откройте **Settings → Pages**.
3. Выберите **Deploy from a branch**, ветку `main`, папку `/ (root)`.
4. После публикации откройте адрес GitHub Pages и выполните жёсткое обновление страницы (`Ctrl+F5`).

## Что исправлено

- Все пути к CSS, JavaScript, манифесту и иконкам сделаны относительными и совместимыми с адресами вида `username.github.io/repository/`.
- Видимые логотипы встроены в `index.html`, поэтому не зависят от загрузки файла из папки `assets`.
- Добавлен файл `.nojekyll`.
- Манифест использует относительные `start_url` и `scope`.

Файл `assets/magnit-mark.png` оставлен для иконки PWA и совместимости.
